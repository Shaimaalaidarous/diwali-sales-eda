# Diwali Sales – Exploratory Data Analysis

## Overview
This project explores customer purchasing behavior during the Diwali season using Python.
The goal is to identify high-value customer segments, key age groups, and top-performing states
that drive overall revenue.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Key Insights
- Female customers contribute higher total revenue, while male customers show higher average order value.
- Customers aged 26–35 represent the most valuable segment.
- A small number of states generate a significant portion of total sales.
- Pareto analysis confirms that a limited percentage of customers drive most of the revenue.

## File
- `diwali_sales_eda.ipynb`: Main analysis notebook

## Dataset
The dataset was sourced from Kaggle (Diwali Sales Dataset) and used for educational and analytical purposes.
